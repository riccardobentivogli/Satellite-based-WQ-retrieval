# Satellite-based Water Quality ML Pipeline

This repository demonstrates a modular workflow for handling patch-based satellite data (e.g., Sentinel-2 5×5 patches) aligned with in-situ environmental measurements.

The focus is on:
- Patch extraction logic
- NPZ dataset structuring
- Feature engineering (center / mean / std)
- Exploratory Data Analysis
- ML/DL analysis to retrieve chlorophyll-a and turbidity
- Clean, reproducible scientific code

---

## Data Policy

This repository does NOT include real satellite or in-situ datasets.

Original research data are private due to collaboration constraints.

To allow reproducibility of the workflow, synthetic dataset generation utilities are included.
